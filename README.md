# 🛋️ Mueblería Web E-Commerce

Plataforma web integral para la venta y gestión de envíos de muebles a domicilio. Este sistema está diseñado para ofrecer una experiencia de compra completa, permitiendo la exploración y personalización detallada de productos.

Es un proyecto Full Stack orientado al comercio electrónico, ideal para demostrar el manejo de interfaces dinámicas, gestión de catálogos y lógica de negocios aplicada a ventas.

## 🚀 Características Principales

- **Catálogo Dinámico:** Exploración de un amplio inventario de muebles con una interfaz amigable.
- **Personalización de Productos:** Sistema que permite a los usuarios configurar sus muebles eligiendo materiales, colores, tamaños y medidas específicas.
- **Gestión de Ventas y Envíos:** Funcionalidad para procesar el carrito de compras y coordinar la logística de entrega a domicilio.
- **Vistas Dinámicas:** Uso del motor de plantillas Handlebars para un renderizado eficiente y escalable desde el servidor.

## 🛠️ Tecnologías y Herramientas

- **Frontend y Vistas:** Handlebars (`.hbs`), HTML5, CSS3, JavaScript.
- **Backend:** Node.js con Express *(Ajustar si usaste PHP, Java o C#)*.
- **Base de Datos:** *(Ej. MySQL o MongoDB - Ajustar según la que hayas implementado)*.
